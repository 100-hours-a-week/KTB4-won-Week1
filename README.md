## 포켓몬 키우기

CLI로 구현한 포켓몬 키우기 게임입니다.

1. 프로그램 실행 시 사용자는 포켓몬의 이름을 지어줍니다.
2. 3~15회 사이에서 포켓몬과 함께 지낼 일 수를 입력합니다.
3. 하루에 하나의 동작이 가능합니다.
    1. 밥 먹기 - 애정도 , 경험치 10 증가
    2. 산책하기 - 애정도, 경험치 20 증가
    3. 사냥하기 - 성공 시 애정도, 경험치 50 증가, 실패 시 애정도 30 감소. 성공 실패는 현재 레벨에 비례하여 랜덤한 확률로 결정

   각 동작 후에는 현재 상태를 출력합니다.

   각 행동은 포켓몬의 경험치와 애정도를 상승시키고, 일정 이상 경험치 획득 시 진화합니다.

4. 정해진 행동이 끝난 후 포켓몬의 레벨, 애정도와 엔딩을 출력한 후 프로그램을 종료합니다.
